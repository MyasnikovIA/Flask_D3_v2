
# Server

Запуск Python скриптов на стороне сервера

## Атрибуты
* name    - наименование компонента
* args - список переменных, который  будет подаватся в качестве аргументов запускаемого скрипта 
 
## Запуск выполнения Python  кода из JS
```
     1.        2.         3.         4.     5.     6.    
  #server(..scriptColl,Form.MyColl,'Arg1','Arg2','Arg3')#

1) #server( )# - ключевое слово, для шифрования имени вызываемой функции 
2) Имя вызываемого компонента cmpServer.  Если перед именем указано символы ".."  значит скрипт находится на этой форме , где вызывается. 
   Если компонент находится на другой форме, тогда указывается путь к другой форме , при этом символ "/" меняются на "."
   Пример: Скрипта c именем "MyPytScript" расположен в папке "/Form/TestScript/main.frm", тогда путь будет такой "TestScript.main.MyPytScript"     
3) Функция которая будет вызываться после выполнения скрипта на стороне сервера. Функция должна иметь один фозвращаемый объект arg
   в который  будут передаватся все инициализированные переменные из Python скрипта
4..6) входящие аргументы , которые передаются из JS в Выполняемый скрипт  
```
 
## Пример

```xml
 
    <cmpScript name="MainScript">
       <![CDATA[
          // Функция вызывает колбэком после  выполнения скрипта на стороне сервера
          Form.MyColl = function(arg) {
              console.log("MyColl", arg );
              console.log("getVar('test')", getVar('test') );
          }
       ]]>
    </cmpScript>
    <cmpServer name="scriptColl" args="testVar1;testvar2;testvar3">
       <![CDATA[
           test = {'test':1111, 'test2':3333, 'arg3':testVar1, 'arg4':testvar2, 'arg4':testvar3}
       ]]>
    </cmpServer>
    
    <cmpButton  caption="Выполнить вызов скрипта на стороне сервера" onclick="#server(..scriptColl,Form.MyColl,'Arg1','Arg2','Arg3')# "/>

```
